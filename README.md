## ğŸ”° About the project

Krushi Arunalu is a web based **Purchase and Sales Management System** for Agri and Crop Products, developed by **CS
Group 19** for Second Year Group Project at UCSC.
There are currently 5 main stakeholders in this system.

- Producer (Farmer)
- Manufacturer (Value-adder)
- Customer
- Agri-Officer
- Admin

### Project at a glance
- Web based system and marketplace to connect farmers, manufacturers/value-adders & end consumers/retailers to streamline the process of producing and selling agri and crop products

- Dynamic crop production suggestions based on land conditions and demand

- Minimises shortage & surplus and maximises gains for all

- Plus, a platform for agri officers to connect with farmers

### â• Prerequisites

### ğŸ”» Downloading

* Clone the code repo using **HTTPS**, SSH or Github CLI or download the zip file

```
git clone https://github.com/UCSC-SYGP-G19/krushi-arunalu.git
```

### ğŸš€ Installing and executing (dev)
Install [XAMPP](https://www.apachefriends.org/it/index.html) for an easy quickstart. [WAMP](https://sourceforge.net/projects/wampserver/) or [MAMP](https://www.mamp.info/en/downloads/) can also be used instead.

1. Move the project folder to the `htdocs` folder in the XAMPP installation directory (or for Apache on Linux, move the project folder to `/var/www/html/`).
2. Create database by importing the latest [sql dump file](https://github.com/UCSC-SYGP-G19/krushi-arunalu/tree/dev/app/db) to your database server.
3. Create `.env` file using the `.sample.env` template and update the required details.
4. Run `composer install` in project root to install the required dependencies and setup git hooks (Make sure you have [composer](https://getcomposer.org/download/) and GNU Bash or [Git Bash](https://github.com/git-for-windows/git/releases/tag/v2.39.2.windows.1) installed).

#### .htaccess files

1. Modify `RewriteBase` of .htaccess file in the root folder to match the name of the project folder inside localhost

```
RewriteBase /<your root folder>/
```

2. Modify `RewriteBase` of .htaccess file in the /public folder to match the name of the project folder inside localhost

```
RewriteBase /<your root folder>/public
```
<br/>
âœ… Open the project in your browser using the URL:

`http://localhost:port/your-root-folder`

For example, if your root folder is `krushi-arunalu` and you are using port 80, the URL will be [http://localhost/krushi-arunalu](http://localhost/krushi-arunalu)
